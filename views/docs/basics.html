The basic commands
One thing all commands have in common: They are all prefixed with !m-, as in !m-start, so they don't double up with other bots! We've tried to make the use of commands as easy as possible, for instance, if you wanted to start the server you would just type !m-start and to stop it again, !m-stop. Wait, did we just learn our first commands? Here's a short overview them again:
Command Role required? Function
!m-start ✓ Starts the webpage mirror
!m-stop ✓ Stops the webpage mirror
