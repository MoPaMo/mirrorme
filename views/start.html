What MirrorMe is not

MirrorMe is not a replace for Discord. If you've got the possibility to use Discord, use it! MirrorMe has just some very rudimentary chat functions for text only (you will also need tp have a Discord server to use it)! It was designed for usecases, when you might need to access data from the outside or for some roleplays where you need to chat anonymously (there are certainly numerous other use cases you will come up - keep in mind it's Open Source so you can always remix it or request new features).
